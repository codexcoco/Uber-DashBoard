# Uber-DashBoard
🚖 Uber Data Analysis Dashboard (Power BI)
📌 Overview

This project presents an interactive Power BI dashboard built using Uber trip data. It focuses on transforming raw data into meaningful insights through data modeling, DAX calculations, and effective visualization techniques. The dashboard enables analysis of ride trends, revenue patterns, and operational performance.

🎥 Project Reference

https://youtu.be/t8eULCaI4x4?si=RgQ3VunyXR8xe9o3

🎯 Objective
Analyze Uber trip data to uncover actionable insights
Build a dynamic and interactive dashboard
Apply data analytics and business intelligence concepts
🛠️ Tech Stack
Power BI
DAX (Data Analysis Expressions)
Power Query
Excel / CSV Dataset

📊 Key Features
Interactive dashboard with filters and slicers
KPI cards for key metrics (Bookings, Revenue, Distance, etc.)
Revenue and trip trend analysis
Vehicle type and payment method insights
Location-based and time-based analysis
Clean and intuitive UI/UX design

🔄 Workflow
Data collection and loading
Data cleaning using Power Query
Data modeling and relationship building
Creating DAX measures and KPIs
Designing dashboard visuals
Adding interactivity (filters, slicers)
